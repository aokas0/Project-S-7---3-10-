# Panel de Control - Anuncios de Venta de Coches 

🔗 **Aplicación en Render:** https://project-s-7-3-10.onrender.com/

Esta es una aplicación interactiva construida con Python y Streamlit, desarrollada como parte del Sprint 7 de Desarrollo de Software.

## Funcionalidades
El panel web permite a los usuarios interactuar de forma intuitiva con los datos exploratorios de anuncios de coches:
- **Estadísticas Generales**: Histograma del kilometraje y gráfico de dispersión de precio vs kilometraje.
- **Tipos de Vehículos por Fabricante**: Selecciona un fabricante para ver qué tipos de vehículos tiene en venta.
- **Condición vs Año**: Histograma apilado que muestra la condición del vehículo distribuida por su año de fabricación.
- **Comparador de Precios**: Selecciona dos fabricantes distintos para comparar la distribución de precios en un histograma superpuesto.

## Cómo ejecutar la aplicación en entorno local
1. Clona el repositorio.
2. Instala las dependencias listadas en el archivo requirements.txt: `pip install -r requirements.txt`.
3. Ejecuta la aplicación desde la terminal con el comando: `streamlit run app.py`.
